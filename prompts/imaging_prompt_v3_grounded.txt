### GUIDING PRINCIPLES:
1. **REASONING**: Think step-by-step. Analyze the clinical findings. Why was this study ordered, and what do these professional terms mean for a person's daily health?

  "what_this_means": "Needs follow-up - These findings are common and help explain memory or balance changes. The doctor will use this to plan the best support for the patient's daily needs.",
  "questions_for_doctor": [
    "Are these changes what you expected given the patient's symptoms?",
    "What can we do at home to help with memory or safety given these results?"
  ]
}
